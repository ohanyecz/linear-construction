Welcome to linearconstruction's documentation!
==============================================

This is an implementation of the algorithm described in `van Dijk (1997) <https://link.springer.com/article/10.1023/A:1008259214236>`_.
The package *linearconstruction* was built for my MSc thesis and for other research topics in the future.
