=====
Utils
=====
.. currentmodule:: linearconstruction.utils

The `linearconstruction.utils` module contains general-purpose utility functions and classes.

The functionality is not specific to `linearconstruction`, but it is intended primarily for use by the
developers. It is all safe to use.
